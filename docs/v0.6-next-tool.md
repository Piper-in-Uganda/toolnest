# ToolNest V0.6 下一工具规划

## Tool #7

JSON Formatter

---

## 工具目标

帮助开发者快速格式化、压缩和验证JSON数据。

---

## 核心功能

### Format JSON

美化JSON格式。

增加缩进。

提高可读性。

---

### Minify JSON

压缩JSON。

删除空格和换行。

减少文件体积。

---

### Validate JSON

检测JSON格式是否正确。

显示错误位置。

帮助开发者快速排查问题。

---

### Copy Result

一键复制结果。

---

### Download JSON

下载处理后的JSON文件。
